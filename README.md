**Workbook Practice Problems**

A static web app that presents interactive study exercises from a JSON file that currently supports multiple choice, with other question types to be added in the future.

An 'example.json' file is provided to show how the template is designed in case users want to customize or create their own JSON file with their custom workbook problems for any miscellaneous subjects.

**Tech Stack**
- HTML
- CSS
- JavaScript
- JSON
